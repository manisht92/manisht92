I'm a **Full-Stack Software Developer** with **12+ years of hands-on experience** crafting robust, scalable, and high-performance applications across web, mobile, and cloud platforms.

My journey started with C# and ASP.NET, and over the years I've grown into a well-rounded engineer who's equally comfortable designing a SQL Server schema, building a React/Angular frontend, shipping a Flutter mobile app, or spinning up containerized microservices on AWS EKS.
